This JavaScript code consists of two parts, both using the DOMContentLoaded event listener to ensure the code runs after the page has fully loaded. The first part handles the functionality of opening a modal with a larger version of a project image when the user clicks on it. It starts by selecting all elements with the class "project-image" and the "modalImage" element by its ID. Then, for each project image, it adds a click event listener that sets the modal image source to the clicked image source. It creates a new Bootstrap modal instance using the "imageModal" element and displays the modal. The second part manages the z-index of carousel items. First, it gets the carousel element with the ID "carouselExampleControls". Then, it adds an event listener for the "slid.bs.carousel" event, which is triggered after a carousel slide transition completes. Within the event listener, it selects the active carousel item and all carousel items. It iterates over each item and sets the z-index to "1" for the active item and "0" for the others. This ensures that the active carousel item is displayed above the others, preventing potential display issues.
